# Vault Schema (Starter)

**This is a skeleton.** Full schema — with reading protocols, linking rules, and agent-routing conventions — is in the Kit.

## Folders

| Folder | Purpose |
|---|---|
| `Areas/<brand>/_context/` | Single source of truth for that brand or area |
| `Projects/` | One flat markdown file per project |
| `Daily Logs/` | One file per day |
| `Meeting Notes/` | Notes from calls |
| `Notes/` | Evergreen reference |
| `Notes/raw/` | Append-only dump zone — agents read, never edit |
| `_System/Raj/` | Your agent's persona files |
| `_System/Templates/` | Obsidian templater templates |
| `MEMORY/` | pai-mobile writes here automatically |
